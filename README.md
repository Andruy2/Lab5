**Краткое описание
Это десктопное приложение на Java для учета клиентов банка и их вкладов. Программа позволяет банковским сотрудникам вести электронную базу клиентов с возможностью хранения данных в SQLite базе данных.

Основные функции
Управление клиентами:

Добавление новых клиентов с указанием ФИО, паспортных данных, типа клиента и суммы вклада

Редактирование информации о существующих клиентах

Удаление клиентов из системы

Просмотр списка всех клиентов в табличном виде

Сортировка клиентов по имени, типу или сумме вклада

Типы клиентов (с разными бонусами):

Обычные клиенты - без бонусов

VIP-клиенты - бонус 10% от суммы вклада

Пенсионеры - фиксированный бонус 3000 рублей

Работа с данными:

Автоматическое сохранение в SQLite базу данных (файл bank_database.db)

Резервное копирование и восстановление базы данных

Экспорт данных в текстовые файлы (TXT/CSV)

Импорт клиентов из текстовых файлов различных форматов

Аналитика:

Статистика по количеству клиентов

Общая сумма всех вкладов

Средний размер вклада

Распределение клиентов по типам**
